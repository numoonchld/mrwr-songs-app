# Song Selector

- Simple React, Redux, and React-Redux App

- `Redux` supplies two named imports:

  - `createStore` function
  - `createReducers` function

- `React-Redux` supplies two named imports:

  - `Provider` component
  - `connect` HOF

- `connect()` used in both
  - Functional Components
  - Class Components
